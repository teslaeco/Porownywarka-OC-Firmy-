# BizInsure – Porównywarka ubezpieczeń firmowych (styl Mubi)

Nowoczesny frontend React inspirowany wyglądem porównywarki mubi.pl, ale dla ubezpieczeń firmowych.

## Demo (lokalnie)

1. Uruchom backend (np. z `/backend/index.js`):
   ```bash
   cd backend
   npm install
   node index.js
   ```
2. Uruchom frontend:
   ```bash
   cd frontend
   npm install
   npm start
   ```
3. Otwórz [http://localhost:3000](http://localhost:3000)

---

## Funkcje

- Kroki porównywarki z paskiem progresu
- Profesjonalny wygląd okna ofert i płatności
- Inspiracja UX/UI stylem Mubi
- Responsywność (działa na mobilu i desktopie)
- Przykładowe loga ubezpieczycieli
